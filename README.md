# Viz


## How to use:

1. Clone project.
2. Navigate to: 
```
./server
```
3. run: 
```
python server.py
```
Notice it run's on port http://127.0.0.1:5000/ (This is the one iOS app simulator is going to use).
4. Navigate to
```
/iOS/VizTest
```
5. open VizTest.xcworkspace
6. run app, you are good to go!
